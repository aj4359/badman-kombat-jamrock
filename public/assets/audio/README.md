# Audio Directory Structure

This directory contains the audio assets for BADMAN KOMBAT.

## Structure:
- `shaw-brothers-intro.mp3` - Iconic Shaw Brothers logo horn blast for VS screen intro
- `bmk-champion-loop.mp3` - Main gameplay soundtrack loop
- `bmk-soundtrack.mp3` - Menu/ambient background music (existing)

## Usage:
- Intro audio plays during VS screen entrance
- Gameplay loop plays during actual combat
- Menu audio plays on main hero screen
